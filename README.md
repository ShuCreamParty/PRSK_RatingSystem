# メモリーコレクト

音ゲーのリザルト画像を読み取り、単曲レートと総合レートを自動計算して管理する Android アプリです。

## このREADMEが対象にしているバージョン

- 対象アプリバージョン: 1.0.0
- 最終更新日: 2026-04-05
- 対象APK: app-release.apk
- 補足: アプリのホーム右下にもバージョン表示があります

## アプリ概要

- 画像フォルダ内のリザルト画像をまとめて読み取ります
- OCRで曲名、難易度、レベル、PERFECT/GREAT/GOOD/BAD/MISS を抽出します
- 抽出結果から単曲レートを算出し、ベスト枠・直近枠・総合レートを更新します
- メニューから「レート対象楽曲」「データ管理」「設定」に移動して運用できます

## APKインストール方法

1. Releases から最新版の app-release.apk をダウンロード
2. 端末設定で「不明なアプリのインストールを許可」を有効化
3. ダウンロードしたAPKを開いてインストール

Releases:

- https://github.com/ShuCreamParty/PRSK_RatingSystem/releases

推奨環境:

- Android 8.0 以上

## 初回設定と基本的な使い方

1. アプリを起動し、設定画面で画像フォルダを選択
2. ホーム画面で「レート算出」をタップ
3. OCRと計算が完了すると、総合レート・ベスト枠・直近枠が更新
4. 必要に応じて「データ管理」から失敗データを手動修正

## レート計算の仕組み

### 1) 単曲レート

判定数から精度を計算します。

- 精度 = (PERFECT×3 + GREAT×2 + GOOD) / ((PERFECT+GREAT+GOOD+BAD+MISS)×3)

単曲レートは以下の分岐です。

- 精度 100%: レベル + 2.0
- 精度 99%以上: レベル + 1.0 + (精度 - 0.99) × 100
- 精度 95%以上: レベル + (精度 - 0.95) × 25
- 精度 95%未満: レベル - (0.95 - 精度) × 50（下限0）

### 2) ベスト枠・直近枠

- ベスト枠: 15曲
- 直近枠: 10曲
- リザーブ枠: 30件（直近候補の母集団）

選抜ルール:

- 同一譜面（曲名+難易度）は、より高い単曲レートの記録を優先
- ベスト枠は、譜面ごとの最高記録から上位15件
- 直近枠は、リザーブ枠内の譜面重複を解消して上位10件

### 3) 総合レート

- ベスト平均 = ベスト枠の単曲レート合計 / 15
- 直近平均 = 直近枠の単曲レート合計 / 10
- 総合レート = (ベスト平均 + 直近平均) / 2

枠が埋まり切っていない場合でも分母は固定です（不足分は0扱い相当）。

### 4) 対象譜面モード

ホームと使用楽曲一覧で対象譜面モードを切り替えできます。

- すべて: 全譜面対象
- MASTERまで: APPEND相当を除外
- APPEND: APPEND相当のみ

APPEND相当の判定:

- 難易度が APPEND
- または MASTER かつ レベル37以上

## 設定でできること

- ユーザー名の変更
- 推しキャラの変更（テーマ配色にも反映）
- テーマ変更（端末準拠 / ライト / ダーク / 各ユニットカラー）
- 画像フォルダの変更
- ゴミ箱(TRASH)画像の一括削除
- DBデータ全削除（画像ファイル自体は残す）

### 削除保護について

- 現在のレート計算に使われている画像は削除不可です
- 画像削除時は、全モード（すべて / MASTERまで / APPEND）で保護判定されます

## データ管理・手動修正

- データ管理画面では FAILED / CALCULATED / TRASH の状態別に確認できます
- 各レコードを開いて手動編集し、保存時にレート再計算できます
- OCR未登録楽曲などで失敗したデータは、FAILED から修正する運用を想定しています

## よくあるケース

- レートが更新されない:
	- 画像フォルダが未設定の可能性があります
	- ホームでレート算出を実行してください
- 読み取り失敗が出る:
	- FAILED一覧で曲名や判定数を手動修正して再計算してください
- 画像が消せない:
	- レート使用中の画像は保護されます
